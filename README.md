## サービス概要
忙しい人向けの「冷蔵庫の中身と購入日を把握する」ためのシンプルな食材管理アプリです。  
冷蔵庫内の食材を簡単に記録・一覧で確認することで、重複購入や食品ロスを防ぎます。  
日々の食事準備や買い物の負担を軽減することを目的としています。

## このサービスへの思い・作りたい理由
平日は仕事で帰宅時間が遅くなることが多く、購入した食材を予定通りに使えないことがよくあります。
気づくと日数が経ってしまい、特に賞味期限や消費期限の記載がない食材については、いつ購入したものか分からなくなり、使うべきかどうか迷ってしまいます。

また、買い物の際にも冷蔵庫の中身を把握しきれず、同じ食材を重複して購入してしまうことが何度もありました。
一方で、冷蔵庫に入っていると思って購入を控えた結果、いざ使おうとしたときには購入から時間が経ちすぎており、結局使えなかったという経験もあります。

食品ロスを減らすために市販の食材管理アプリも試しましたが、入力項目が多く、手間に感じて継続できませんでした。
「最低限の入力で、今の自分にちょうどいい管理ができるサービスがあればいいのに」という思いから、このサービスを作りたいと考えました。

## ユーザー層について
- 仕事と家事・育児を両立している忙しい社会人  
- 日々の買い物や献立決めに負担を感じている人 
- 食材管理がうまくいかず、食品ロスが気になる人

特に、帰宅時間が遅く、平日にまとめ買いをすることが多い人を主な想定ユーザーとしています。

## サービスの利用イメージ
ユーザーは購入した食材を簡単に登録し、一覧で確認できます。
買い物前に在庫状況や購入日を把握することで、重複購入や不要な買い足しを防ぐことができます。
その結果、食品ロスの削減につながるだけでなく、日々の買い物に対する心理的な負担の軽減にもつながります。

## サービスの差別化ポイント・推しポイント
既存の食材管理アプリは高機能なものが多い一方で、入力や管理の手間が大きく、継続しづらいと感じていました。
本サービスでは、レシピ提案や栄養管理などの機能はあえて行わず、「今ある食材を把握する」ことに機能を限定しています。  
忙しい日常の中でも無理なく使い続けられる点が最大の推しポイントです。

## 機能候補
### MVPリリース時に作っていたい機能
- ユーザー登録・ログイン機能  
- 野菜マスター一覧表示  
  - 特に管理が難しい野菜を対象に、あらかじめ用意した一覧から選択できるようにします
- 野菜マスターから選択して在庫登録  
  - 購入日を登録し、自分の在庫として管理します  
- 在庫一覧表示
  - 登録した野菜を一覧で確認できます  
- 野菜マスターに存在しない野菜の追加 
  - 一覧にない野菜はユーザー自身が追加できるようにします
- よく使う野菜のお気に入り登録  
  - 頻繁に購入する野菜をお気に入りとして登録し、一覧の上部に表示できるようにします  

### 本リリースまでに作っていたい機能
- 対象食材の拡張  
  - 野菜以外の食材（肉・魚・冷凍食品など）にも対応  
- カテゴリ（肉・野菜・冷凍など）管理  
- 消費期限の簡易管理  
  - 野菜は購入日から目安の期限を自動提案し、手動で変更可能  
  - 期限表示のある食材は、実際の期限を手動で入力可能  
- 消費期限が近い食材の表示  
- 検索・絞り込み機能
  - 食材マスターに対して、名前やカテゴリで検索・絞り込みができる機能
